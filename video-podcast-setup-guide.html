Hybrid Recording Approach: Best of Both Worlds

Eliminate daily card dumping while maintaining failsafe backup recording.

Direct-to-Computer Recording with Hardware Backup

The Smart Hybrid Approach

Record directly to your computer for convenience while maintaining hardware recording as a failsafe backup.

Primary Recording (Failsafe)
  • ATEM ‚Üí SSD: Always recording ISOs as backup
  • PodTrak P8 ‚Üí SD Card: Always recording multitrack audio
  • These run independently of your computer
  • 100% reliable, no dependency on software
Secondary Recording (Convenience)
  • DaVinci Resolve: Recording for immediate access
  • All ISO video feeds recorded directly
  • All audio tracks from PodTrak via USB
  • If it crashes, you still have card/SSD recordings
  • If it works, you skip the card dumping step

Workflow Steps

  1. Start hardware recording first:
    • Hit record on ATEM (to SSD)
    • Hit record on PodTrak P8 (to SD)
  2. Then start DaVinci recording:
    • This gives you instant file access
    • But hardware backup is running
  3. If computer is stable:
    • Use the DaVinci files
    • Format cards/SSD for next day
  4. If computer crashes:
    • You have full backup on cards/SSD
    • Only lose convenience, not content
Benefits:
  • 99% of time: save 10-15 minutes daily
  • 1% of time: fall back to cards
  • Zero risk of losing recordings
  • Instant access to files when it works
  • Peace of mind with dual recording
Requirements:
  • Powerful computer (32GB RAM minimum)
  • Fast storage (SSD for recording)
  • Stable system (no other apps running)
  • DaVinci Resolve Studio ($295)

DaVinci Resolve Setup Guide

Initial DaVinci Configuration

One-Time Setup:
  1. Connect Hardware:
    • ATEM ‚Üí Computer via USB-C
    • PodTrak P8 ‚Üí Computer via USB-C
  2. DaVinci Preferences:
    • Go to: Preferences ‚Üí System ‚Üí Audio
    • Set "Audio Device" to "PodTrak P8"
    • Channel Configuration: Manual
    • Input Channels: 12
    • Output Channels: 2
  3. Create Project Template:
    • Timeline Frame Rate: Match your cameras
    • Resolution: 1920x1080 or higher
    • Audio Sample Rate: 48kHz

PodTrak P8 USB Setup

On the PodTrak P8:
  1. Connect USB-C to your computer
  2. Press MENU ‚Üí Audio Interface ‚Üí On
  3. Select USB Audio Mode: "Multi Track" (not "Stereo Mix")
  4. Sample Rate: 48kHz (match your video)
Channel Routing:
Channels 1-6: Your XLR inputs (mics)
Channels 7-8: USB return from computer
Channels 9-10: Sound pad outputs
Channels 11-12: Main mix

Recording Session Setup

For Each Recording:
  1. Create New Project from your template
  2. Go to Fairlight page
  3. Add Audio Tracks:
    • Right-click in mixer ‚Üí Add Track ‚Üí Audio
    • Add 4 Mono tracks (one per mic)
  4. Route Each Track:
    • Click "Input" ‚Üí Patch Input/Output
    • Source: Audio 1, Audio 2, etc. (from P8)
    • Destination: Track 1, Track 2, etc.
  5. Configure Video Capture:
    • Media page ‚Üí Capture
    • Select ATEM as video source
    • Enable "Capture all inputs as individual clips"
  6. Arm tracks for recording (R button)
  7. Start Recording in capture window

Troubleshooting

No audio showing in Resolve?
  • Check P8 is in Multi Track mode
  • Restart Resolve after connecting P8
  • Check Windows/Mac audio settings
Only getting 2 channels?
  • P8 might be in Stereo Mix mode
  • Change to Multi Track mode
Crackling/dropouts?
  • Increase buffer size in Resolve
  • Check USB cable quality
  • Close other applications
  • Use dedicated USB bus

Audio-Video Integration: The AFV Challenge

Understanding Audio Follow Video (AFV)

The Problem: In a multi-person podcast, when you switch to a close-up of Person B, you want:

  • Microphone B to be prominent in the mix
  • Other microphones to automatically lower (duck)
  • Smooth, professional transitions
  • No manual audio mixing during recording

Without AFV: Audio and video are separate systems, requiring manual sync and mixing in post-production.

Option 1: ATEM Mini Pro ISO + Intelligent Control

Setup: Use the basic ATEM with added automation

Hardware:

  • ATEM Mini Pro ISO: $895
  • Zoom PodTrak P8: $300
  • Computer running Companion software
  • Total added cost: ~$1,195

How it Works:

  1. Physical Connections: 
    Mics ‚Üí PodTrak P8 ‚Üí Main Mix ‚Üí ATEM Audio Input
                               ‚Üì
                             SD Card (Multitrack backup)
  2. Automation via Companion:
    • Install Bitfocus Companion (free)
    • Create macros linking camera switches to audio changes
    • When switching to Camera B:
      • Companion sends MIDI to PodTrak
      • Mic B level increases
      • Other mics duck by preset amount
  3. Workflow:
    • Operator uses ATEM buttons to switch cameras
    • Companion automatically adjusts audio mix
    • Both changes are recorded
Pros:
  • Keeps equipment costs lower
  • Maintains separate audio recording
  • Can be customized extensively
Cons:
  • Requires initial setup and programming
  • Another software layer to manage
  • Less reliable than hardware AFV

Complete Signal Flow

Option 1: Basic Setup (ATEM Mini Pro ISO)

Audio Path

  1. SM7B Microphones ‚Üí XLR cables ‚Üí PodTrak P8
  2. PodTrak P8 ‚Üí Main Mix Out ‚Üí ATEM Audio Input
  3. PodTrak P8 ‚Üí SD card (multitrack recording for post)
  4. Optional: Companion software controls audio levels via MIDI

Video Path

  1. Cameras ‚Üí HDMI cables ‚Üí ATEM Mini Pro ISO
  2. ATEM ‚Üí USB-C SSD (all ISO recordings + program)
  3. ATEM ‚Üí HDMI out to confidence monitor
  4. All video files share same audio mix (no AFV)

Post-Production Requirements

  • Manual audio sync required
  • Use multitrack from PodTrak for individual control
  • More time needed for audio mixing

Option 2: Professional Setup (ATEM Television Studio HD8 ISO)

Integrated Audio/Video Path

  1. SM7B Microphones ‚Üí XLR ‚Üí ATEM Television Studio HD8
  2. Cameras ‚Üí HDMI ‚Üí ATEM Television Studio HD8
  3. Internal Fairlight mixer processes with AFV
  4. Recording outputs:
    • Each ISO file has AFV-processed audio
    • Separate audio ISO files preserved
    • Program file with live mix

Post-Production Benefits

  • Each camera file already has appropriate audio mix
  • Can use as-is for quick turnaround
  • Individual audio tracks still available for fine-tuning
  • AutoPod works seamlessly with pre-mixed audio

Post-Production Workflow (Both Options)

  1. Import all ISO files into Premiere Pro template
  2. Create multicam sequence (or use AutoPod)
  3. For Option 1: Sync with multitrack audio from PodTrak
  4. For Option 2: Audio already embedded correctly
  5. Apply MOGRTs for graphics
  6. Export master and social cuts

Daily Operation Workflow

Pre-Production Checklist

  • Update CSV file with guest names/episode info
  • Check all cable connections
  • Format recording media (SSD and SD cards)
  • Test all microphone levels
  • Verify camera framing
  • Load intro/outro music on PodTrak

During Recording

  • Start recording on PodTrak P8
  • Start recording on ATEM Mini Pro ISO
  • Monitor audio levels throughout
  • Mark good takes/segments
  • Note any technical issues for post

Post-Production Pipeline

  1. File Transfer (5 min)
    • Copy files from SSD to edit station
    • Copy audio from SD card
    • Create project folder from template
  2. Rough Edit (15-30 min with AutoPod)
    • Import to Premiere template
    • Run AutoPod for initial edit
    • Sync high-quality audio
  3. Fine Edit (30-45 min)
    • Review AutoPod decisions
    • Add graphics via MOGRTs
    • Color correction if needed
    • Audio sweetening
  4. Export (15-20 min)
    • Master file (4K or 1080p)
    • YouTube version
    • Podcast platforms (audio only)
    • Social clips (vertical)
